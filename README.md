# Warp Template Apple Multiplatform

Generate the Xcode project with `xcodegen generate`.

Open `WarpTemplateAppleMultiplatform.xcodeproj` in Xcode 16+ and run the four platform schemes:

- `WarpTemplateiOS`
- `WarpTemplateiPadOS`
- `WarpTemplatemacOS`
- `WarpTemplatewatchOS`

The `FoundationModelsBridge` package is intentionally scaffolded only and exposes a no-op `availability()` API.
